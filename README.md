## 🏵 Introduction

**MiniToast** is an Android toast view implementation for iOS.

![preview](https://i.loli.net/2018/03/21/5ab2368f47ca2.png)

## 📋 Requirements

- iOS 9.0+
- Xcode 9.0+ with Swift 5.0+

## 📲 Installation

`MiniToast` is available on [CocoaPods](https://cocoapods.org):

```ruby
use_frameworks!
pod 'MiniToast'
```
